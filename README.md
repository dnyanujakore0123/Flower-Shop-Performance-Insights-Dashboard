# Flower-Shop-Performance-Insights-Dashboard
This project is a Power BI dashboard that analyzes flower shop sales and customer behavior.  It provides insights into total orders, client trends, and product demand.  The dashboard helps understand which flowers are most popular and how sales vary over time.

🎯 Objective

To analyze flower shop sales data and identify demand patterns.

To track customer orders and purchasing behavior.

To support better inventory and sales decisions.

🛠️ Tools Used

Power BI

📊 Dashboard Features

Total Orders, Unique Clients, and Demand Score KPIs

Monthly trend analysis of orders

Client-wise order distribution

Demand score analysis by flower type

Interactive filters for flower name and client name

📌 Key Insights

Rose has the highest demand among all flowers

A few clients contribute to most of the total orders

Sales show fluctuations across different months

Demand varies significantly by flower type
